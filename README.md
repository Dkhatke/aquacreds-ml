# AquaCreds ML – Blue Carbon MRV System

This repository contains the machine learning pipeline used in the AquaCreds project (Smart India Hackathon 2025 Finalist).

## Overview
The model analyzes satellite imagery to estimate mangrove canopy metrics and biomass for Blue Carbon monitoring.

## Features
- Satellite band extraction
- Feature engineering
- Biomass prediction models
- Dataset preprocessing pipeline
